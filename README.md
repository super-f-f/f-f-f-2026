## ex04
摸一下触摸引脚，LED亮起并保持长亮；松开手再摸一下，LED熄灭。
## ex05
LED持续呈呼吸灯效果。
用户每触摸一次引脚，LED“呼吸”的节奏就会发生改变（如从缓慢呼吸变为急促呼吸，共有三个明显的速度级别）。
## ex06
两个LED灯呈现平滑交替的渐变闪烁效果，一个变亮的同时另一个逐渐变暗，过渡非常柔和。
## ex07
手机或电脑浏览器打开ESP32网页，页面上出现一个滑动条。拖动滑动条时，开发板上的LED亮度会随着滑动条的位置实时、平滑地改变。
## ex08
 未布防时触摸引脚无反应；在网页点击布防后，一旦有人触碰引脚，LED开始高频闪烁锁定报警；只有通过网页点击撤防，LED才会熄灭并重置状态。
## ex09
网页上显示着一个实时跳动的数字。当手逐渐靠近ESP32引脚时，网页上的数字实时变小；手指离开时，数值恢复。实现类似仪器仪表的数据实时监控面板。
